# 주요 법령 전문 오프라인 폴백

법제처 Open API 키(`LAW_OC`)가 아직 발급되지 않았거나 네트워크가 막힌 환경에서 **시연·강의·기능 실습**을 계속하기 위한 폴백이다. 별도 설치 없이 JuriSupport 플러그인에 포함된 `references/statutes/` 전문 스냅샷을 읽어 사용한다.

## When to use

- 사용자가 "API 키가 아직 없다", "시연이라 법제처 키 발급 전이다", "오프라인 법령 팩으로 진행"이라고 말한 경우
- `korean-law` MCP가 세션 도구 목록에 없거나, 법제처 API 호출이 키/네트워크 문제로 실패했는데 사용자가 실습·데모 목적임을 밝힌 경우
- `/jurisupport:brief-protocol` 또는 `/jurisupport:mock-hearing` 중 법령 조문 예시가 필요하지만 정식 검증 경로를 사용할 수 없는 경우

## Do not use

- 실제 사건 서면, 제출 예정 문서, 의뢰인에게 제공할 최종 의견서의 법령 검증 완료 근거
- 이 팩에 없는 조문을 추측해서 보충하는 용도
- 판례 검색·판례 원문 확인. 이 팩은 판례를 포함하지 않는다.

## 사용 규칙

1. 먼저 `references/statutes/INDEX.md`를 열어 포함 범위와 기준일을 확인한다.
2. 답변에는 반드시 `오프라인 스냅샷(기준일: 2026-06-14)`이라고 표시한다.
3. 실무 제출 가능성 있는 결과물에는 `제출 전 korean-law MCP 또는 국가법령정보센터에서 최신 조문 재검증 필요` 문구를 붙인다.
4. 포함되지 않은 법령·조문은 "오프라인 팩에 없음"이라고 말하고, 키 발급 후 `korean-law` MCP 재조회로 넘긴다.
5. 조문 원문을 직접인용할 때는 해당 법령 전문 파일의 텍스트와 글자 단위로 맞춘다.
6. 생성한 결과물이 실습용이 아니라 실제 제출용으로 전환되면, Phase 3 검증을 다시 실행하고 온라인 검증 실패 상태로 제출 가능 판정을 내리지 않는다.

## 포함 법령

- Core: 대한민국헌법, 민법, 민사소송법, 형법, 형사소송법, 상법
- Special criminal: 특정범죄가중법, 특정경제범죄가중법, 폭력행위처벌법, 성폭력처벌법, 아청성보호법, 스토킹처벌법, 마약류관리법, 도로교통법, 교통사고처리특례법, 가정폭력처벌법, 성매매처벌법, 아동복지법, 공직선거법, 조세범처벌법, 부정경쟁방지법, 저작권법, 변호사법, 자본시장법, 정보통신망법, 개인정보보호법, 전자금융거래법, 범죄수익은닉규제법

## 출력 예시

```md
[오프라인 스냅샷 | 기준일 2026-06-14 | 실습용]

민법 제750조는 "고의 또는 과실로 인한 위법행위로 타인에게 손해를 가한 자는 그 손해를 배상할 책임이 있다."고 정합니다.

※ 이 조문은 JuriSupport 오프라인 법령 전문 스냅샷 기준입니다. 실제 제출 전에는 korean-law MCP 또는 국가법령정보센터로 최신 조문을 재검증해야 합니다.
```
