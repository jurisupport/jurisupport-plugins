# lbox.kr 판례 검색 워크플로우 (상세)

> 자동화 X. 사용자가 직접 검색하고 클로드코드가 결과를 분석하는 흐름.

---

## 전제 조건

- lbox.kr 유료 계정 보유
- 브라우저(Chrome/Safari/Edge)
- 사건폴더 구조 확립 (예: `~/사건/사건명_사건번호/`)

---

## 표준 워크플로우 5단계

### Step 1. 사건 쟁점 정리 (클로드코드)

사건폴더에서 클로드코드 세션을 열고:

```
이 폴더의 사건기록을 보고 핵심 쟁점을 3개로 정리해줘.
각 쟁점에 대해 검색해볼 만한 키워드도 함께 제안해줘.
```

출력 예시:
```
쟁점 1. 소멸시효 완성 vs 채무승인
  키워드: "소멸시효 채무승인", "시효이익 포기", "시효 완성 후 변제"

쟁점 2. 차용증의 법적 효력
  키워드: "차용증 진정성립", "기존 채무 확인"

쟁점 3. 일부 변제의 의미
  키워드: "일부변제 채무승인", "변제 의사 묵시적 표시"
```

### Step 2. lbox에서 직접 검색

1. https://lbox.kr 로그인
2. 검색바에 키워드 입력 (위에서 추천받은 것)
3. 결과 페이지에서:
   - **대법원 판결** 우선 (선례 가치)
   - **선고일 최근** 우선 (변경된 법리 반영)
   - **관련성 높은 순** (lbox 자체 정렬)
4. 후보 판결 3~5건 선정

### Step 3. 판결문 PDF 다운로드

각 판결의 상세 페이지에서:
- "**PDF 다운로드**" 버튼 (lbox 공식 제공 기능)
- 파일명에 사건번호 포함됨

저장 위치:
```
사건명_사건번호/
└── 참고판례/
    ├── 대법원_2013다64793.pdf
    ├── 대법원_2018다234567.pdf
    └── 서울고법_2020나123456.pdf
```

### Step 4. 클로드코드에 일괄 분석 요청

```
참고판례/ 폴더의 PDF 3건을 모두 읽고,
본 사건 쟁점 1(소멸시효 채무승인)에 어떻게 적용되는지
각 판결의 사건번호·선고일·법리를 표로 정리해줘.
그 다음 가장 유리한 인용 문구 3개를 추출해줘.
```

### Step 5. 준비서면에 인용

클로드코드가 정리한 인용 문구를 준비서면에 반영. 단:

- **직접인용 (`"..."`)은 PDF 원문과 글자 단위 일치 검증**
- 일치 안 되면 간접인용(요지)로 변경
- 사건번호는 반드시 원문 그대로
- 본문에는 "**판결**"로 표현 (`판례`는 X)

## 검색 키워드 패턴 (자주 쓰는 조합)

| 분야 | 키워드 |
|---|---|
| 시효·소멸 | `[시효 완성] [채무승인]`, `[시효이익 포기]`, `[일부변제 시효중단]` |
| 임대차 | `[보증금 반환] [동시이행]`, `[임대차 갱신거절]`, `[전세금 우선변제]` |
| 노동 | `[부당해고] [정당한 이유]`, `[징계해고 절차]`, `[퇴직금 산정]` |
| 사해행위 | `[채권자취소] [사해의사]`, `[수익자 악의 추정]` |
| 사기 손해배상 | `[기망행위] [재산상 처분]`, `[변제 의사 부존재]` |
| 의료 | `[설명의무] [의료과실]`, `[전원의무]` |

## 주의사항 모음

1. **lbox 자동화 절대 금지** — 계정 영구 차단 + 약관 위반
2. **PDF 재배포 금지** — 본인 사건 사용 한정
3. **사건번호 인용 시 실존 재확인** — 클로드가 자동으로 가져온 번호도 lbox에서 다시 검색해 일치 확인
4. **하급심 판결 → 항소심·상고심 결과 별도 확인**
5. **변경된 법리 주의** — 선고일 10년 이상 지난 판결은 후속 판결로 변경되었을 가능성 점검
