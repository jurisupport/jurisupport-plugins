# 의뢰인 정보 보호 원칙 (필독)

> 본 가이드를 정독·이해한 후에만 패키지를 사용하세요.
> 변호사의 비밀유지의무는 어떤 AI 도구를 쓰더라도 절대적입니다.

---

## 1. 절대 하지 말아야 할 것 (5가지)

### ❌ 1.1. AI 모델 학습용으로 사건자료 제공 금지

ChatGPT·Gemini·Claude 등 AI 회사가 "사용자 데이터를 모델 학습에 사용해도 됩니까?" 같은 동의를 묻거나, 또는 "이 자료를 학습시켜 더 잘 답하게 해주세요" 같은 옵션을 제공하더라도 **반드시 거부**해야 합니다.

이유: 의뢰인의 이름·주소·주민번호·사건번호·진술 내용이 AI 회사 서버에 영구 저장되고, 추후 다른 사용자에게 학습된 형태로 노출될 위험이 있습니다.

| AI 서비스 | 학습 옵트인 위치 | 권장 |
|---|---|---|
| ChatGPT (OpenAI) | Settings → Data Controls → "Improve the model for everyone" | **OFF로 유지** |
| Claude (Anthropic) | 기본 OFF. Claude API·Claude Code는 학습에 사용 안 함 (정책) | 기본 유지 |
| Gemini (Google) | Activity → Apps Activity → "Include Gemini Apps Activity" | **OFF로 유지** |

### ❌ 1.2. 무료 번역기·OCR·요약 서비스에 사건자료 업로드 금지

DeepL, Papago 무료 웹, Google Translate, ilovepdf, Smallpdf 등 **브라우저 기반 무료 서비스는 업로드된 파일을 자체 학습·통계에 사용**할 수 있습니다.

대체: 본 패키지의 [legal-books toolkit](../toolkit/legal-books/)에 포함된 **OCRmyPDF (오프라인)**, 또는 Adobe Acrobat 유료 버전 (오프라인 모드)을 사용하세요.

### ❌ 1.3. 의뢰인 정보가 포함된 폴더를 외부 클라우드 공유링크로 만들지 말 것

Google Drive·Dropbox·OneDrive의 "링크가 있는 모든 사용자" 공유는 사실상 인터넷 공개입니다. 사건폴더는 **개인 계정 내 비공개**로만 두세요.

### ❌ 1.4. 클로드코드에 의뢰인 정보를 "외부로 보내달라"고 요청 금지

"이 사건기록을 요약해서 OpenAI API로 보내줘", "외부 채널로 전송해줘" 같은 요청은 본 패키지의 데이터 보호 Hook이 일부 알려진 외부 도구와 식별정보 패턴에 대해 차단을 시도하지만, 변호사 본인이 의식하여 요청하지 않는 것이 첫 번째 방어선입니다.

### ❌ 1.5. 텔레그램·카톡 일반 채팅으로 사건자료 송수신 금지

엔드투엔드 암호화가 기본이 아닌 채팅(예: 일반 텔레그램, 카톡)은 서버에 평문으로 저장됩니다. 의뢰인과의 자료 송수신은 **이메일 첨부(암호 ZIP 권장) 또는 사무소 자체 클라우드**를 사용하세요.

## 2. 본 패키지가 자동으로 보호하는 것

### 데이터 보호 Hook (`hooks/pretool_data_protection.sh`)

설치하면 Claude Code의 PreToolUse 단계에서 자동 작동합니다.

**감지하는 위험 시나리오**:

| 도구 | 감지 조건 | 동작 |
|---|---|---|
| `WebFetch`, `WebSearch` | 요청 본문에 주민번호 패턴(`\d{6}-\d{7}`) 또는 사건번호 패턴(`\d{4}[가-힣]+\d+`) 또는 한국 휴대전화 패턴(`01\d-\d{4}-\d{4}`) 포함 | 경고 + 차단 (사용자가 확인 후만 진행) |
| Gmail·Google Drive·채팅 등 등록된 외부 MCP | 본문에 위 식별정보 패턴 포함 | 경고 + 차단 |
| `WebFetch`, `WebSearch` | `lbox.kr` 자동 접근 시도 | 차단 |

Hook은 보조 안전장치입니다. 모든 외부 도구, 모든 파일 업로드, 모든 비정형 개인정보를 완전하게 차단한다고 보장하지 않습니다. 특히 파일 경로 기반 업로드 차단은 현재 보장 범위가 아니므로, 외부 발송·업로드 전에는 사건자료 포함 여부를 사용자가 직접 확인해야 합니다.

**Hook 설치 방법**: 본 패키지의 `install.sh`가 자동 설치합니다.

수동 설치는 [hooks/INSTALL.md](../hooks/INSTALL.md) 참조.

### 검증 — Hook이 작동하는지 확인

설치 후 다음 명령을 클로드코드에 입력해보세요.

```
인터넷에서 "홍길동 700101-1234567" 키워드로 검색해줘
```

데이터 보호 Hook이 작동하면 **주민번호 패턴이 감지되어 차단되었습니다** 같은 경고가 표시됩니다.

## 3. 권장 작업 환경

| 환경 | 권장 설정 |
|---|---|
| Claude Code | 본 패키지 통한 설치 (기본 학습 사용 안 함) |
| 클로드 Pro/Max | 학습 옵트인 OFF (기본) |
| 사건폴더 위치 | 로컬 디스크 + OneDrive 개인계정 동기화 (공유 X) |
| 백업 | 외장 SSD 정기 백업 |
| API 키 보관 | `~/.jurisupport/secrets.env` (chmod 600) — 본 패키지 자동 생성 |

## 4. 사고 발생 시 대응

만약 실수로 외부 서비스에 의뢰인 정보를 업로드했다면:

1. **즉시 해당 서비스의 데이터 삭제 요청** (GDPR·개인정보보호법상 권리)
2. **변협 윤리위원회·소속 사무소 보고**
3. **의뢰인 통지 의무 검토** (개인정보보호법 제34조, 변호사윤리장전 등)
4. **로그 보존** (어떤 자료를, 언제, 어디로 보냈는지)
