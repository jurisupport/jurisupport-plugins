# 법고을 (lx.scourt.go.kr) 판례·법령 검색 워크플로우 (상세)

> 대법원 도서관 운영, 무료, 비로그인 가능. lbox 무료 대안.

---

## 사이트 개요

- **URL**: https://lx.scourt.go.kr
- **운영**: 대법원 도서관
- **비용**: 완전 무료
- **로그인**: 선택 (비로그인도 검색·PDF 다운로드 가능)

## 다루는 자료

| 탭 | 내용 |
|---|---|
| 판례 | 대법원·고등법원·하급심 판결문 |
| 법령 | 법률·시행령·시행규칙 본문 |
| 규칙·예규·선례 | 대법원 규칙, 등기예규 등 |
| 문헌 | 법학 논문·판례 평석 |
| 해외판례 | 미국·중국·일본 등 주요 외국 판결 (영문) |

## 표준 워크플로우

### Step 1. 검색 키워드 정리 (클로드코드)

`lbox 워크플로우`와 동일. 사건폴더에서 쟁점·키워드 추출.

### Step 2. 법고을에서 직접 검색

#### A. 통합검색 (가장 빠름)

1. https://lx.scourt.go.kr 접속
2. 상단 통합검색바에 키워드 입력
3. 결과 카테고리(판례/법령/문헌)에서 원하는 자료 선택

#### B. 판례 전용 검색 (상세 필터)

1. 상단 **"판례"** 탭
2. 검색바 + 필터:
   - 법원 (대법원, 고등법원, 지방법원)
   - 사건 종류 (민사, 형사, 행정, 가사 등)
   - 선고일 범위
3. 결과 목록 → 사건 선택

#### C. 사건번호 직접 조회

검색바에 사건번호 그대로 입력: `2013다64793`

### Step 3. PDF 다운로드

판결문 상세 페이지에서:
- "**PDF 다운로드**" 버튼
- 사건폴더의 `참고판례/`에 저장

### Step 4. 클로드코드 분석

`lbox 워크플로우`와 동일.

## 법령 검색 (보너스 기능)

법고을은 **법령 본문도 검색** 가능. 다만 한국법령정보센터(law.go.kr)가 더 표준적이라, 본 패키지의 [korean-law MCP](../plugins/jurisupport/)를 우선 사용 권장.

### 법고을 법령 검색 사용 시점

- korean-law MCP가 작동 안 할 때 비상용
- 시행령·시행규칙까지 한 화면에서 보고 싶을 때
- 부칙 / 별표 / 별지 서식을 봐야 할 때

## 검색 팁

| 시나리오 | 방법 |
|---|---|
| AND 검색 | 공백으로 구분 (`시효 채무승인`) |
| OR 검색 | `|` 또는 사이트별 옵션 |
| 정확한 문구 | 큰따옴표 (`"시효이익의 포기"`) |
| 제외 검색 | `-` (`형사 -양형`) |
| 사건종류 좁히기 | 좌측 필터 활용 |
| 최신 판결만 | 선고일 정렬 또는 기간 설정 |

## 법고을 vs lbox 비교

| 항목 | 법고을 | lbox |
|---|---|---|
| 비용 | 무료 | 유료 |
| 검색 정확도 | 보통 | 우수 (AI 요약 포함) |
| 하급심 자료 | 일부 | 다수 |
| 사용 편의성 | 기본 | 모던 UI |
| 자동화 위험 | 동일 (금지) | 동일 (금지) |
| 비고 | 정부 공식 | 민간 |

→ **추천 운영 방식**:
1. **콜드스타트 / 비용 부담 큰 변호사**: 법고을만 사용
2. **본격 실무**: lbox 유료 가입 + 법고을은 보조
