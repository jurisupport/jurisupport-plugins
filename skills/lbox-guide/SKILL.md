# lbox.kr 판례 검색 가이드 스킬

본 스킬은 lbox 검색 절차를 안내하고, 다운로드한 결과를 클로드코드가 분석하도록 돕습니다.

## When to use

- "lbox에서 어떤 키워드로 찾아야 할지 모르겠다"
- "lbox에서 받은 판결문 PDF를 분석해줘"
- "이 사건 쟁점에 맞는 판결 검색 전략 알려줘"

## 사용 절차

### Step 1. 검색 키워드 추출 (클로드코드)

사용자가 사건 쟁점이나 법리 질문을 던지면, 클로드코드가 **lbox 검색에 적합한 키워드 후보 3~5개**를 제시합니다.

좋은 키워드 예시:
- **법리 단어 + 쟁점 단어** 조합: "소멸시효 채무승인", "임대차 동시이행"
- **법령 조문 명시**: "민법 162조 1항", "근로기준법 23조"
- **사실관계 핵심**: "차용증 작성 시효이익 포기"

### Step 2. lbox.kr에서 검색

1. https://lbox.kr 접속하여 로그인
2. 상단 검색바에 키워드 입력
3. 결과 목록에서 사건 선택
4. 판결문 페이지 진입 → "**PDF 다운로드**" (lbox 자체 기능)
5. 다운로드된 PDF를 사건폴더의 `참고판례/` 하위에 저장

### Step 3. 클로드코드에 분석 요청

저장한 PDF를 클로드코드에 분석시킵니다.

```
참고판례/대법원_2013다64793.pdf를 읽고
본 사건 쟁점(시효 완성 후 채무승인)에 어떻게 적용되는지 정리해줘.
```

클로드코드가 판결문을 읽고 (kordoc MCP 또는 내장 PDF 읽기로) 적용 분석·인용 문구 추출 등을 수행합니다.

## 검색 키워드 추천 가이드

| 쟁점 유형 | 추천 키워드 조합 |
|---|---|
| 시효 항변 반박 | `[시효 완성] [채무승인] [시효이익 포기]` |
| 임대차 분쟁 | `[보증금] [동시이행] [임대인 변경]` |
| 부당해고 | `[정당한 이유] [징계해고] [부당해고]` |
| 사해행위 | `[채권자취소] [사해의사] [악의 추정]` |
| 손해배상 | `[과실상계] [위자료] [정신적 손해]` |

## 결과물 저장 권장 구조

```
사건명_사건번호/
└── 참고판례/
    ├── 대법원_2013다64793.pdf       (lbox 다운로드)
    ├── 대법원_2013다64793_요약.md   (클로드코드 분석 결과)
    └── 검색이력_2026-05-17.md       (어떤 키워드로 무엇을 찾았는지)
```

## 주의사항

- **다운로드 PDF는 본인 사용 한정** (재배포·공유 금지)
- **무료 대체**: 법고을(lx.scourt.go.kr) — `beopgoeul-search` 스킬 참조
- 판결 인용 시 반드시 사건번호 실존 재확인 (PDF 받은 그 사건이 lbox에 실재함을 확인)
