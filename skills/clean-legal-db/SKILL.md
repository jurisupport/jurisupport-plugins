# 클린 법률 DB 검색 (clean-legal-db)

법령·판례 근거가 필요하면 이 로컬 DB를 키워드 검색한다. **수록 자료는 전부 저작권법 §7(법령·판결) 또는 공공누리 제1유형(국가법령정보센터)으로 자유이용·배포가 보장된 자료**다.

## 사전 확인

DB가 설치돼 있어야 한다(약 235MB, 설치 시 GitHub Release에서 1회 다운로드).

```bash
ls ~/clean-legal-db/clean_legal.db
```

- 파일 없음 → 사용자에게 설치 안내: `bash ~/jurisupport-plugins/toolkit/clean-legal-db/install.sh`

## 사용 방법

```bash
python3 ~/clean-legal-db/search.py "검색어"
```

- `--type 법령` 또는 `--type 판례` : 종류 한정
- `--top N` : 결과 개수(기본 8)
- `--json` : JSON 출력(프로그램 연계용)

### 예시
```bash
python3 ~/clean-legal-db/search.py "위법수집증거 증거능력"
python3 ~/clean-legal-db/search.py "배임"                 # 2글자도 가능
python3 ~/clean-legal-db/search.py "유류분" --type 판례 --top 5
python3 ~/clean-legal-db/search.py "채무불이행 손해배상" --type 법령
```

## 수록 범위 (총 18,150여 건)
- **법령(현행 전문)** 약 11,000조문 / 56종: 헌법·민법·형법·상법·민소·형소·민집·가사소송·행정소송·행정심판·행정절차·행정기본·국가배상·저작권·상표·특허·개인정보보호·정보공개·주택임대차·상가임대차·집합건물·부동산등기·도로교통·근로기준·노동조합·산재보상·국세기본·소득세·부가가치세 등
- **자치법규(조례)** 약 3,100조문
- **판례** 1,140건: 민·형·소송·행정·가사 주요 쟁점 대법원·하급심(판시사항·판결요지·참조조문·전문)
- **행정심판 재결** 792건: 운전면허·영업정지·정보공개·산재 등(청구취지·주문·재결요지·이유)
- **조세심판원 결정** 645건: 양도세·부가세·상속·증여 등 세목(주문·재결요지·관련법령·이유)
- **노동위 판정** 391건: 부당해고·부당노동행위 등(판정사항·판정요지·내용)
- **헌재 결정** 317건 · **행정규칙(고시·훈령·예규)** 254건 · **개인정보위 결정** 148건 · **법제처 법령해석** 354건

`--type` 값: `법령` `자치법규` `판례` `헌재결정` `행정심판재결` `조세심판` `노동위판정` `행정규칙` `개인정보위결정` `법령해석`

## 검색 방식
- SQLite FTS5(trigram) 키워드 검색 + 2글자 쿼리용 LIKE 폴백. **API 키/인터넷 불필요.**

## 답변 시 주의
- 결과에 표시된 **인용(법령 조문명 / 판례 사건번호)을 그대로 출처로 인용**할 것.
- 판례 사건번호의 최종 진위는 사용자(변호사)가 확인. DB에 없는 조문/판례는 "DB 미수록"으로 명시.
- 이 DB는 현행 법령·공개 판례의 일부 집합이며 전체 법령/판례를 대체하지 않는다.
- 출처 표시: 국가법령정보센터(법제처). 상세 근거는 `~/clean-legal-db/COPYRIGHT.md`.
